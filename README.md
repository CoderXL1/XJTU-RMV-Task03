# XJTU-RMV-Task03
### 视觉组第三次作业
#### Ceres 应用：拟合小球弹道

**拟合模型——假设小球弹道满足：**

$$
x(t)=x_0+{v_{x0}\over k}(1-e^{-kt})
$$

$$
y(t)=y_0+{(v_{y0}+{g\over k})\over k}(1-e^{-kt})-{g\over k}t
$$

其中进行了屏幕坐标转换，将 <u>以左上角为原点，向右为 $x^+$，向下为 $y^+$ 的坐标系</u> 转换为了 <u>以左下角为原点，向右为 $x^+$，向上为 $y^+$ 的坐标系</u>。

参考数据：帧高度 $720$，帧宽度 $1280$。

-------------

拟合结果：

```
vx0: 244.564
vy0: 340.795
k: 0.0404434
g: 486.6
```
